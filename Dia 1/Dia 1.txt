Usando Inputs em terminal em Java:
Para usar o input em Java precisa colocar no começo do codigo:
	import java.util.Scanner;
	
Quer seria a importação da bliblioteca, para utilizar ela em codigo, é necessario chamar ela na void main, dessa forma:
	Scanner scanner = new Scanner(System.in);

Colocando isso, eu podeirei usar o "scanner.next" dizendo que vai ser um input, temos:
	 scanner.nextLine(); 	-	Usado para ler string
	 scanner.nextInt(); 	-	Usado para ler numero inteiros
	 scaneer.nextDouble();	- 	Usado para ler numeros reais

Para usar em uma variavel nova, pode ser usada dessa forma:
	{Tipo} {nomeVariavel} = scaneer.next{tipoNext}();
 
Quanto terminar, precisa fechar o input, usando:
	scanner.close();

Exemplo:
	import java.util.Scanner;

	public class Main{
		public static void main(String[] args){
			// Usando Input em Java
			Scanner scanner = new Scanner(System.in);
			System.out.println("Digite o seu segundo Nome:");
			String segundoNome = scanner.nextLine();
			System.out.println("Seu segundo nome e: "+segundoNome);
			scanner.close();
			System.out.println("Saido do SCANNER");
		}
	}

*Terminal*

Usando em estrutura de IF:
	import java.util.Scanner;

	public class Main{
		public static void main(String[] args){
			// Usando IF e Else operadores:
			Float nota1 = 4.5f;
			Float nota2 = 4.5f;
			Float nota3 = 4.5f;
			Float nota4 = 4.5f;
			Float media = ((nota1 + nota2 + nota3 + nota4)/4);
			
			System.out.println("Media do aluno: "+media);
			
			if (media < 5f){
				System.out.println("Aluno Reprovado");
			} else if (media > 5f || media < 7){
				System.out.println("Aluno em Recuperacao");
			} else {
				System.out.println("Aluno Aprovado");
			}
		}
	}
